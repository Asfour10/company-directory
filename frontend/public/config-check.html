Frontend Configuration Check

This page helps verify which backend URL the frontend is configured to use.

Expected Backend URL:

https://company-directory-backend-o3xm.onrender.com

Instructions:

  1. Open browser Developer Tools (F12)
  2. Go to Console tab
  3. Type: localStorage and press Enter
  4. Check if there's an old backend URL stored
  5. Clear it with: localStorage.clear()
  6. Try login again
